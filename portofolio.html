Get weather
For the "Get Weather" project, I used the information provided on Weather Underground by making a request to the weather API. The user has two possibilities to find the local Weather or to know the weather in any other city. To find the current location I used the geolocation API. Since the answer was asynchronous   ....see more 
I wrapped it in a Promise. To check the weather in other cities the user must fill the Country, Region and Location inputs, but it is not mandatory to fill them all. The parameter values for location accepted by Weather API are only the country and location. If there are several places with the same name, weather API does not provide us with the data needed but provide us in response those locations. To solve this problem I decided, in the weather API URL, for the value of the location parameter to use the geographical position of the one entered. To get latitude and longitude of the desired town I made a call to the Google Maps Geocoding API that provides the geographic position of the town entered by the user in related fields. This value, saved in a variable, I pass to the weather API URL, by chaining this call (weather APY call) with Google Maps Geocoding API call. In this way, the user gets the exact weather information for the desired location.    ....see less
